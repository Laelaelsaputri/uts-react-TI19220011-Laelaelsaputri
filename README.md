DESKRIPSI APLIKASI
Aplikasi ini adalah Task Management App berbasis web yang dikembangkan menggunakan React.js dengan Redux Toolkit untuk manajemen state dan React Router DOM untuk navigasi halaman. Aplikasi memungkinkan pengguna untuk:
1. Menambah, melihat, mengedit, dan menghapus tugas/proyek.
2.  Menyimpan informasi seperti judul, deskripsi, kategori, dan lampiran file PDF atau gambar.
3.  Menyimpan data secara lokal di browser menggunakan localStorage, sehingga tetap dapat diakses meski halaman direfresh tanpa server eksternal.
4.  Aplikasi ini cocok digunakan untuk mengelola tugas atau proyek secara digital dengan antarmuka sederhana dan interaktif.

CARA MENJALANKAN APLIKASI
1.  Buka aplikasi di browser.
2.  Tambah tugas baru dengan mengisi judul, deskripsi, kategori, dan lampiran jika perlu.
3.  Lihat daftar tugas di halaman utama.
4.  Edit atau hapus tugas dengan tombol yang tersedia.
5.  Semua data tersimpan otomatis di browser dan tetap ada saat direfresh.
